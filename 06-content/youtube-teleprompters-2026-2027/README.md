# YouTube-телесуфлеры 2026-2027

Пакет содержит 51 телесуфлер: по одному на каждый выпуск календаря.

Формат каждого выпуска:

- 6 страниц;
- печатная разметка 11 pt;
- сухой разговорный текст;
- минимум наречий и прилагательных;
- без сложных расчетов в речи;
- все денежные примеры только в евро;
- философские выпуски не называют философа в кликабельном заголовке и в spoken-тексте; имя и литература вынесены в блок "Не читать" для описания.

Файлы:

- `episodes-md/` - отдельные Markdown-тексты.
- `episodes-html/` - отдельные HTML-телесуфлеры с 11 pt.
- `all-teleprompters-11pt-print.html` - общий файл для печати или PDF.
- `all-teleprompters-11pt.pdf` - общий PDF на 306 страниц после печати из HTML.
- `manifest.csv` - индекс выпусков и путей.
- `quality-audit.json` - техническая проверка страниц, объема и запрещенных маркеров.
