# Brand OS Дмитрия Ефремова

Это единое место для управления личным брендом, направлениями, философскими заметками, метриками, фото и публичными каналами.

Главная идея: ты пишешь в чат обычным языком, а я обновляю конкретные файлы здесь. Markdown нужен для смысла и текстов, YAML/CSV - для структурированных данных, которые удобно сверять и дополнять.

## Как пользоваться из чата

Можно писать так:

- "добавь направление: ..."
- "обнови брендбук: тон должен быть строже / теплее / инженернее"
- "запиши метрику Instagram за 2026-06-06: ..."
- "добавь философа: имя, идеи, как связано с моим подходом"
- "положи это фото в approved и подпиши для сайта"
- "собери из Brand OS пост для Instagram / сценарий YouTube / блок на сайт"
- "покажи все незаполненные места в Brand OS"

## Структура

- `00-system` - правила, источники, команды управления.
- `01-profile` - личное позиционирование, ссылки, контакты.
- `02-brand` - брендбук, голос, визуальная система.
- `03-directions` - направления деятельности и продуктовые офферы.
- `04-philosophy` - философы, концепты, связи с бизнесом и продуктами.
- `05-metrics` - метрики, снимки состояния, правила обновления.
- `06-content` - сайт, YouTube, Instagram, контентные рубрики.
- `07-assets` - фото, логотипы, визуальные материалы.
- `08-inbox` - входящие идеи из чата до сортировки.
- `09-operations` - журнал изменений и регулярные процедуры.
- `dashboard` - локальный HTML-пульт для обзора календаря, рубрик, философов, метрик, ассетов и команд.

## Текущий статус

Первая версия создана на основе публичного сайта и ссылок, которые ты дал в чате. После этого в Brand OS импортированы материалы из прошлых чатов Codex:

- финальный брендбук и короткая fact-only спецификация;
- пакет `31` аналитический философ с биографиями и книгами;
- EMS-сортировка философов по темам продукта и бизнеса;
- курс/каталог по `20` MVP-метрикам и продуктовой аналитике;
- Instagram grid references;
- финальный пакет фото-референсов Дмитрия.

Импортированные материалы лежат в разделах `imported`, а управленческие индексы - рядом с основными файлами.

Локальный пульт управления: `dashboard/index.html`.
