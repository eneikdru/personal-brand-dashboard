# Knowledge Base (База Знаний) - FBUN CRIE

Проект по созданию централизованной базы знаний для образовательного центра ФБУН ЦНИИ Эпидемиологии Роспотребнадзора.

## Обзор проекта
Цель проекта — создание единого информационного пространства для поддержки образовательного процесса по программам ординатуры, аспирантуры и ДПО. База знаний строится на платформе Moodle (sdo.crie.ru) с интеграцией в существующую электронную информационно-образовательную среду (ЭИОС).

## Основные характеристики
- **Платформа:** Moodle (sdo.crie.ru)
- **Целевые группы:** Администраторы, Контент-менеджеры, Преподаватели, Обучающиеся (ординаторы, аспиранты).
- **Ключевые интеграции:** ЭИОС, Teachbase, Telegram, мессенджер Max.
- **Срок реализации:** декабрь 2026 г. (с промежуточными этапами).

## Структура репозитория
- `docs/` — Проектная документация и техническое задание.
- `tasks/` — Бэклог задач и план внедрения.
