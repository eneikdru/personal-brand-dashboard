# Архитектура и иерархия Базы Знаний (Moodle)

## 1. Структура разделов (Категории)
Иерархия курсов/категорий в Moodle для обеспечения логичной навигации:

- **1. Нормативная база**
  - 1.1. Федеральные документы (ФГОС, Приказы Минобрнауки/Роспотребнадзора)
  - 1.2. Локальные нормативные акты (ЛНА) ЦНИИ Эпидемиологии
  - 1.3. Изменения в законодательстве (Мониторинг)
- **2. Прием и движение контингента**
  - 2.1. Приемная кампания (по учебным годам: 2024-25, 2025-26, 2026-27)
  - 2.2. Перевод, отчисление и восстановление
  - 2.3. Академические отпуска
  - 2.4. Прикрепление соискателей
- **3. Аттестации и контроль качества**
  - 3.1. Текущий контроль и промежуточная аттестация
  - 3.2. Государственная итоговая аттестация (ГИА) — Ординатура
  - 3.3. Итоговая аттестация — Аспирантура
  - 3.4. Кандидатские экзамены
  - 3.5. ВСОКО и апелляции
- **4. Стипендии и материальная поддержка**
  - 4.1. Положения и регламенты
  - 4.2. Работа стипендиальной комиссии
- **5. Учет результатов и документы**
  - 5.1. Индивидуальный учет результатов (ИУР)
  - 5.2. Личные дела и персональные данные
  - 5.3. Выдача свидетельств и справок
- **6. Образовательный процесс и практика**
  - 6.1. Организация процесса (смешанное/сетевое обучение, ИУП)
  - 6.2. Практическая подготовка и НИР
  - 6.3. Элективные и факультативные дисциплины
- **7. Управление и кадры**
  - 7.1. Положения об отделах и советах (УМС)
  - 7.2. Индивидуальные планы работы (ИПР) ППС/НПР
- **8. Платные образовательные услуги**
- **9. Справочная информация**
  - 9.1. Глоссарий и сокращения
  - 9.2. Контакты и FAQ
- **10. Архив документов**

## 2. Метаданные карточки документа
Для каждого загружаемого файла (ресурс «Файл» или «Страница») заполняются следующие поля:

| Поле | Описание | Пример |
|---|---|---|
| **Тип документа** | Категория документа | Положение, Порядок, Шаблон, Проект |
| **Наименование** | Полное название документа | Положение о стипендиальном обеспечении |
| **Период действия** | Учебный год или «Бессрочно» | 2026-2027 |
| **Программа** | На кого распространяется | Аспирантура, Ординатура, Все |
| **Процедура** | Бизнес-процесс | Прием, Аттестация, Практика |
| **Версия** | Номер редакции | v2.1 |
| **Статус** | Текущее состояние | Действующий, Архив, Проект |
| **Ответственный** | Лицо, отвечающее за актуальность | Иванов И.И. (Зав. отделом) |
| **Теги** | Ключевые слова для поиска | ГИА, стипендия, приём-2026 |

## 3. Логика версионности и архивирования
1. **Замена документа:** При публикации новой версии старая карточка получает статус «Архив» и перемещается в категорию «10. Архив документов».
2. **Маркировка:** В названии архивного документа добавляется префикс `[АРХИВ]` и указывается дата вывода из эксплуатации.
3. **Связи:** В описании актуального документа дается ссылка на предыдущую версию в архиве (и наоборот).

## 4. Использование инструментов Moodle
- **Ресурс «Файл»:** Для неизменяемых документов (PDF-сканы приказов).
- **Ресурс «Страница»:** Для инструкций и FAQ с возможностью быстрого поиска.
- **Элемент «Книга»:** Для многостраничных регламентов (например, Руководство по практике).
- **Элемент «Глоссарий»:** Единый справочник терминов и сокращений.
- **Блоки уведомлений:** Вывод последних обновлений на главную страницу базы знаний.
