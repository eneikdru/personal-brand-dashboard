# Фото

Здесь хранится фотоархив личного бренда.

## Папки

- `inbox` - новые фото без разбора.
- `approved` - фото, которые можно использовать.
- `archive` - устаревшие или неиспользуемые фото.

## Правило именования

`YYYY-MM-DD_context_role_version.ext`

Примеры:

- `2026-06-06_portrait_site_v01.jpg`
- `2026-06-06_youtube_cover_business-philosophy_v01.png`
- `2026-06-06_instagram_mvp_post_v01.jpg`

## Каталог

Все фото регистрируются в `photo-catalog.csv`.

Минимум для каждого фото:

- путь к файлу;
- статус;
- назначение;
- настроение;
- права использования;
- заметки.
