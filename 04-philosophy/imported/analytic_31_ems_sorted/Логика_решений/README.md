# Логика понятий

**Влияние на менеджмент:** Менеджмент как дисциплина точных различений: что именно утверждается, на каком основании и что из этого следует.

**Влияние на EMS:** EMS получает язык решений: меньше тумана, меньше декоративных слов, больше проверяемых формулировок.
