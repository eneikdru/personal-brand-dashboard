# Модели и причинность

**Влияние на менеджмент:** Менеджмент как проверка причинных механизмов, а не поклонение красивым диаграммам.

**Влияние на EMS:** EMS получает MVP как прибор: продуктовая версия должна измерять, что реально меняет поведение.
