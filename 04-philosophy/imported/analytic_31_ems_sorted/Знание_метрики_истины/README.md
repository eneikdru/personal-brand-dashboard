# Знание и ошибка

**Влияние на менеджмент:** Менеджмент как управление статусом знания: мнение, гипотеза, факт, проверка, пересмотр.

**Влияние на EMS:** EMS получает метрики истины: система должна отличать знание от уверенного шума.
