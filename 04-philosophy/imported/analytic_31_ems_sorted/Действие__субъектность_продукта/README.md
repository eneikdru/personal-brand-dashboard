# Действие и мастерство

**Влияние на менеджмент:** Менеджмент как организация намеренного действия, ответственности и передачи мастерства.

**Влияние на EMS:** EMS получает субъектность: команда должна не исполнять шум, а понимать действие, за которое отвечает.
