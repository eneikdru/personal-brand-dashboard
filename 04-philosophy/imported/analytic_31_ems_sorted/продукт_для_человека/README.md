# Мышление и среда

**Влияние на менеджмент:** Менеджмент как проектирование среды, в которой человек думает, ошибается, замечает и действует.

**Влияние на EMS:** EMS получает продукт для живого человека: меньше перегруза, больше опор для внимания и решения.
