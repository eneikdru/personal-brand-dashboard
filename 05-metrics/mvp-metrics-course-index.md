# MVP Metrics YouTube Course

Статус: `imported`

Полный пакет: `imported/mvp_metrics_youtube_course`.

## Главная логика

MVP начинается не с метрик, а с гипотезы: какую ценность и для кого проверяем. Хорошая метрика имеет числитель, знаменатель, период, сегмент и решение.
