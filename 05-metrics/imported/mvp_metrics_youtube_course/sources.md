# Sources and Reference Anchors

Эти материалы курса написаны как практические конспекты, а не академический обзор. Исторические якоря и формулы сверялись с общеупотребимыми источниками:

- [NPS: original HBR article by Frederick F. Reichheld, December 2003](https://hbr.org/2003/12/the-one-number-you-need-to-grow)
- [Net Promoter System by Bain, background and history](https://www.netpromotersystem.com/about/)
- [AARRR / Startup Metrics for Pirates, Dave McClure deck](https://www.slideshare.net/slideshow/startup-metrics-for-pirates-presentation/629833)
- [Lean Startup Co: Eric Ries MVP definition](https://leanstartup.co/resources/articles/what-is-an-mvp/)
- [Lean Analytics, One Metric That Matters, O'Reilly](https://www.oreilly.com/library/view/lean-analytics/9781449335687/ch06.html)
- [Amplitude North Star Framework](https://amplitude.com/books/north-star/about-north-star-framework)
- [Stripe guide to SaaS metrics](https://stripe.com/us/resources/more/essential-saas-metrics)
- [ChartMogul SaaS metrics library](https://chartmogul.com/saas-metrics/)

## Как использовать источники
- Для выпусков про MVP и validated learning: Lean Startup.
- Для North Star и фокусной метрики: Lean Analytics и Amplitude.
- Для AARRR: Dave McClure / Startup Metrics for Pirates.
- Для SaaS economics: Stripe и ChartMogul.
- Для NPS: HBR и Bain.
