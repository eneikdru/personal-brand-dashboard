# Команды управления из чата

Эти команды не обязательны дословно, но такой формат ускорит работу.

## Направления

`добавь направление: [название]; аудитория: [...]; оффер: [...]; доказательства: [...]`

`обнови направление [id]: [что изменить]`

`покажи карту направлений`

## Брендбук

`обнови брендбук: [правило / формулировка / запрет / визуальный принцип]`

`сделай голос бренда более [строгий / философский / инженерный / человеческий]`

`проверь текст на соответствие брендбуку: [текст]`

## Философы

`добавь философа: [имя]; идеи: [...]; связь с моим подходом: [...]`

`собери философскую рамку для темы: [тема]`

`свяжи [философ] с направлением [id]`

## Метрики

`запиши метрику: дата: YYYY-MM-DD; канал: [...]; метрика: [...]; значение: [...]`

`покажи последние метрики`

`найди слабое место в воронке`

## Фото и ассеты

`зарегистрируй фото: файл: [...]; назначение: сайт / Instagram / YouTube / аватар / обложка`

`подбери фото для направления [id]`

`покажи фото без статуса approved`

## Контент

`собери пост из направления [id]`

`сделай сценарий YouTube на основе философа [имя]`

`покажи телесуфлер для выпуска #[номер]`

`обнови телесуфлер для выпуска #[номер] по новым вводным: [что изменить]`

`адаптируй этот текст под сайт / Instagram / YouTube`
