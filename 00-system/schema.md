# Схема данных

## Базовые сущности

| Сущность | Где хранится | Для чего |
|---|---|---|
| Профиль | `01-profile/profile.md`, `00-system/brand-os.yml` | кто ты, как себя описываешь, основные ссылки |
| Бренд | `02-brand/brandbook.md` | голос, визуал, запреты, смысловая рамка |
| Направление | `03-directions/*.md` | оффер, аудитория, доказательства, контентные углы |
| Философ | `04-philosophy/philosophers.md` | идеи, цитаты, связь с бизнесом и продуктами |
| Метрика | `05-metrics/metrics-registry.csv`, `05-metrics/snapshots/*.md` | измерение роста и эффективности |
| Фото | `07-assets/photos/photo-catalog.csv` | учет файлов, назначений, статусов и прав |
| Контент | `06-content/*.md` | рубрики, каналы, идеи, готовые черновики |

## Статусы

- `draft` - набросок.
- `active` - можно использовать.
- `approved` - утверждено Дмитрием.
- `needs_validation` - нужно уточнить.
- `archived` - не использовать без отдельного запроса.
